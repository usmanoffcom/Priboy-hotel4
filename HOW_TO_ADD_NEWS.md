# Как добавить новость отеля

## Где редактировать

Все новости хранятся в файле: `lib/news-data.ts`

## Структура новости

```typescript
{
  id: "1",                    // Уникальный ID (можно использовать номер)
  slug: "nazvanie-novosti",   // URL-дружественное название (латиница, дефисы)
  title: "Заголовок новости", // Заголовок новости
  excerpt: "Краткое описание", // Краткое описание (отображается в списке)
  content: "<p>Полный текст новости в HTML формате</p>", // Полный текст с HTML
  image: "/slides/1.jpg",      // Путь к изображению (из папки public/)
  date: "15 января 2026",      // Дата в формате "DD месяца YYYY"
  author: "Администрация",     // Автор (опционально)
  featured: true,             // Показывать в featured (опционально)
}
```

## Пример добавления новости

Откройте файл `lib/news-data.ts` и добавьте новость в массив `newsPosts`:

```typescript
export const newsPosts: NewsPost[] = [
  {
    id: "1",
    slug: "spetsialnoe-predlozhenie-na-leto-2026",
    title: "Специальное предложение на лето 2026",
    excerpt: "Раннее бронирование со скидкой до 30%! Забронируйте номер на летний сезон заранее и получите лучшую цену.",
    content: `
      <p>Гранд Отель & SPA Прибой предлагает специальные условия раннего бронирования на летний сезон 2026 года.</p>
      
      <h2>Условия акции:</h2>
      <ul>
        <li>Скидка до 30% при бронировании до 1 марта 2026</li>
        <li>Бесплатная отмена до 7 дней до заезда</li>
        <li>Дополнительные бонусы для постоянных гостей</li>
      </ul>
      
      <p>Спешите забронировать номер по выгодной цене!</p>
      <p><a href="/booking">Забронировать номер</a></p>
    `,
    image: "/slides/1.jpg",
    date: "15 января 2026",
    featured: true,
  },
  // Добавьте здесь другие новости...
]
```

## Важные моменты

1. **Slug** должен быть уникальным и содержать только латинские буквы, цифры и дефисы
2. **Content** поддерживает HTML разметку (заголовки, списки, ссылки, изображения)
3. **Image** - путь относительно папки `public/`
4. **Date** - используйте формат "DD месяца YYYY" на русском языке
5. Новости автоматически сортируются по дате (новые первыми)

## После добавления

1. Сохраните файл
2. Закоммитьте изменения: `git add lib/news-data.ts && git commit -m "Добавлена новость: название"`
3. Запушьте: `git push origin main`
4. Задеплойте: `./deploy.sh`

## Где отображаются новости

- **Список новостей**: `/news`
- **Отдельная новость**: `/news/[slug]`
- Новости автоматически добавляются в sitemap.xml

## Примеры HTML в content

```html
<h2>Заголовок</h2>
<p>Обычный текст</p>
<ul>
  <li>Элемент списка</li>
</ul>
<p><a href="/booking">Ссылка</a></p>
<p><strong>Жирный текст</strong></p>
```
