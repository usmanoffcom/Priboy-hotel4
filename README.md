# Гранд Отель & SPA Прибой - Документация проекта

## 📌 Основная информация

- **Название**: Гранд Отель & SPA Прибой
- **Домен**: `priboy-spa.ru`
- **Технологии**: Next.js 16, React, TypeScript, Tailwind CSS
- **Сервер**: `89.23.102.48`
- **Порт**: `3002`
- **Удаленная директория**: `/var/www/priboy-spa.ru`
- **Systemd сервис**: `priboy-spa-ru.service`

## 🎯 Особенности проекта

### Бронирование
- **Интеграция с TravelLine** для бронирования
- Виджет бронирования на странице `/booking`
- Нет кастомной формы (используется TravelLine)

### Контент
- Отель с собственным SPA
- Акцент на SPA-комплексе
- Роскошный дизайн и контент
- Подчеркивание премиальности

## 🚀 Деплой

### Скрипт деплоя
Используется `../deploy-priboy-spa-ru.sh` из корня репозитория.

### Команда деплоя
```bash
cd /Users/renatusmanov/Priboy4
export DEPLOY_PASSWORD='wR_DUF3Ays3kVu'
bash deploy-priboy-spa-ru.sh
```

### Что делает скрипт
1. Синхронизирует файлы через `rsync`
2. Устанавливает зависимости (`pnpm install`)
3. Собирает проект (`pnpm build`)
4. Устанавливает права доступа для `www-data`
5. Настраивает systemd сервис
6. Настраивает Nginx
7. Устанавливает SSL сертификат (если нужно)
8. Перезапускает сервис

## 📝 Важные файлы

### SEO и метаданные
- `app/sitemap.ts` - Динамический sitemap для поисковых систем
  - Включает статические страницы
  - Включает динамические страницы номеров
  - Включает страницы блога
  - Включает страницы SPA-услуг

### Компоненты
- `components/spa-preview.tsx` - Превью SPA-комплекса
  - Акцент на премиальности

  - Виды бань
  - Массажи
  - Косметология
  - Процедуры

- `lib/blog-data.ts` - Данные блога
  - SEO-оптимизированные описания
  - Категории статей
