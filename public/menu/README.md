# Изображения меню ресторана

Пожалуйста, скопируйте сюда изображения из Яндекс.Диска:

1. **Изображения блюд** (для раздела "Фирменные блюда"):
   - `barabulka.jpg` - Черноморская барабулька
   - `shashlik.jpg` - Шашлык из баранины
   - `khachapuri.jpg` - Хачапури по-аджарски
   - `creme-brulee.jpg` - Крем-брюле

2. **Изображения меню** (PDF или изображения):
   - `menu-main.jpg` или `menu-main.pdf` - Основное меню
   - `menu-lunch.jpg` или `menu-lunch.pdf` - Меню ланчей
   - `menu-kids.jpg` или `menu-kids.pdf` - Детское меню

После копирования файлов код автоматически подхватит их.
